### AIModelOps

Full-stack prompt management platform built for automating LLM workflows.


### Structure

```tree
AIModelOps/
├── frontend/
├── runner/
├── cli/
├── db/
│   ├── prompts.db
│   └── schema.sql
├── README.md
├── requirements.txt
├── package.json

```


### Stack

- **Frontend**: React (w/ Chakra UI), Tauri, TypeScript
- **Scripting**: Node.js + Python
- **CI/CD**: To be added


### Setup

```bash
cd frontend
npm install
npm run dev

```


### Instructions

```bash
pip install -r requirements.txt

```
